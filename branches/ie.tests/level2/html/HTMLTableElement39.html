The insertRow() method inserts a new empty table row. If index is -1 or equal to the number of rows, the new row is appended. Retrieve the second TABLE element and invoke the insertRow() method with an index of negative one. The number of rows in the TBODY section before insertion with an index of negative one is two. After the new row is inserted the number of rows in the TBODY section is three.
