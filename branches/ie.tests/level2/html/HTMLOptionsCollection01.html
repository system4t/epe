description

An HTMLOptionsCollection is a list of nodes representing HTML option element. The length attribute specifies the length or size of the list. Retrieve the first SELECT element and create a HTMLOptionsCollection of the OPTION elements. Check the size of the length of OPTION elements.

http://www.w3.org/TR/DOM-Level-2-HTML/html#HTMLOptionsCollection-length
