description

The namedItem(name) method retrieves a node using a name. It first searches for a node with a matching id attribute. If it doesn't find one, it then searches for a Node with a matching name attribute, but only on those elements that are allowed a name attribute. Retrieve the first FORM element and create a HTMLCollection by invoking the elements attribute. The first SELECT element is further retrieved using the elements id attribute.
