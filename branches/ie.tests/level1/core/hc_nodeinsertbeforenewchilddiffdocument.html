description

The "insertBefore(newChild,refChild)" method raises a WRONG_DOCUMENT_ERR DOMException if the "newChild" was created from a different document than the one that created this node. Retrieve the second employee and attempt to insert a new child that was created from a different document than the one that created the second employee. An attempt to insert such a child should raise the desired exception.
