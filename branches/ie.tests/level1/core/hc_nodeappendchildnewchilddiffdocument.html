description

The "appendChild(newChild)" method raises a WRONG_DOCUMENT_ERR DOMException if the "newChild" was created from a different document than the one that created this node. Retrieve the second employee and attempt to append a node created from a different document. An attempt to make such a replacement should raise the desired exception.
